Basic SQL Simulator created in C++. Currently you can create and drop tables. You can add columns, insert values and display the whole table.

To run , clone the repository, go into the directory and run the command in the terminal:

`g++ -I include src/main.cpp src/Coloana.cpp src/Tabela.cpp src/BazaDeDate.cpp src/Interogare.cpp src/Rand.cpp -o test.exe`

You need to have GCC installed to compile the project.

For the list of commands and how to use them : `/test.exe --help`

Project in development, will be updated soon with more commands.
